# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Be patient. Be welcoming, especially to newcomers.
- Critique ideas, not people.
- Assume good faith; ask clarifying questions before assuming the worst.
- No harassment, no hate speech, no personal attacks. Period.

## Reporting

If you experience or witness behavior that violates the Covenant — in issues, PRs, discussions, or any project space — email **conduct@uaip.dev**. Reports are confidential.

The full Contributor Covenant 2.1 text governs this project's interactions. The link above is authoritative.
